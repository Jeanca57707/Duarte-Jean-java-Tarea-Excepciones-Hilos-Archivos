# Duarte-Jean-java-Tarea-Excepciones-Hilos-Archivos
Tarea de repaso de excepciones, archivos e hilos.


# Explicación de la tarea
 
 Esta tarea consta de 5 ejercicios, cada uno tiene todas sus clases en sus carpetas correspondintes; ejemplo:
 el ejercicio uno completo se encuentra en la carpeta llamada "ejercicioUno", así con las demás.

 Los primeros 2 ejercicios tratan del manejo de Hilos en java, el tercero y cuarto ejercicio van sobre manejo de excepciones, y el quinto va de manejo de archivos. He aquí una breve explacación de cada uno.

## Explicacion de cada ejercicio

## Ejercicio 1

Simulador de Descargas.

Un programa que simula la descarga de tres archivos al mismo tiempo.
Cada archivo se ejecuta en un hilo independiente y muestra su progreso en pantalla hasta completar la descarga.

## Ejercicio 2

Carrera de corredores. 
Un programa que simula un carrera de 4 corredores diferentes utilizando hilos. Al final muestra quíen ganó ka carrera.

## Ejercicio 3

Validar edad. Un programa que maneja excepciones para ver si es mayor o menor de edad. Si es menor de edad, lanzará una
excepción personalizada al usuario.

## Ejercicio 4

 Calculadora. Una calculadora que puede sumar, restar, multiplicar y dividir. Esta enviará un mensaje al usuario si
 intenta dividir entre 0, o si ingresa texto en vez de un número.

## Ejercicio 5

 Manejo de archivos. Un programa que te permite registrar actividades en un archivo llamado "actividades.txt",
 y que tambien te permite leer lo quese guardo en el archivo. La información del archivo no se eobreescribe ni se borra
 al momento de agregar nuevas actividades.

 ## Como ejecutar cada ejercicio

 Dentro de la consola, tendrá que moverse entre las diferentes carpetas que se encuentran en el repositorio usando el comando: ("cd nombreDeLaCarpeta"). Estando dentro de cualquiera de las carpetas/directorios, ejecute los siguientes comandos: primero "javac Main.java" y luego "java Main.java". Ahí el programa empezará a ejecutarse en su consola.

 ## Importante
    Debe tener Java o JDK para poder ejecutar los últimos dos comandos mencionados.
